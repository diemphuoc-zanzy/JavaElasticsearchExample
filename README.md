## BUILD AND RUN ELASTICSEARCH

### Step 1
- Download Elasticsearch for window [click here](https://www.elastic.co/guide/en/elasticsearch/reference/current/zip-windows.html)
- Extract file .zip
- Go to file \bin of file extracted
